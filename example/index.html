The links below show how to use this demo, it supports 3 parameters:
geohash
geohash accuracy, valid values are: geohash_1, geohash_2, geohash_3 (default), geohash_4 or cellid
fq
Solr filter query, examples are: taxon_key:1, basis_of_record:OBSERVATION. Consult the Solr schema for more details (default value is empty, i.e.: no filter.)
solr
URL to the target Solr server, default values is http://prodsolr1-vh.gbif.org:8983/solr/uat_occurrence/
